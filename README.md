# OpenStereo

This is an awesome open source toolbox for stereo matching.

## Supported Methods:

- [x] BM
- [x] SGM(T-PAMI'07)
- [x] GCNet(ICCV'17)
- [x] PSMNet(CVPR'18)
- [x] StereoNet(ECCV'18)
- [x] CFPNet(ICIEA'19)
- [x] HSMNet(CVPR'19)
- [x] GwcNet(CVPR'19)
- [ ] STTR(ICCV'21)
